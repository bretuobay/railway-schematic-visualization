# Storybook

This Storybook is the interactive companion to the VitePress docs. Use it for quick visual inspection of core rendering, adapter behavior, and ecosystem package examples.

## Included Example Sets

- Core rendering and SVG output
- Framework adapter integration surfaces
- Ecosystem production features such as theming, i18n, SSR, and canvas rendering

Run `npm run storybook` after installing Storybook locally.
