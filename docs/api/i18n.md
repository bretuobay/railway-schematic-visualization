# I18n API

Package: `@rail-schematic-viz/i18n`

Use this package when you need localized labels, number and date formatting, or RTL-aware UI behavior.

## Key Exports

- `I18nManager`
- locale contracts and translation types

## Responsibilities

- locale registration
- translation lookup with fallback handling
- number and date formatting
- UI direction and RTL support
