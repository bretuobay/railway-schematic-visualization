# Context Menu API

Package: `@rail-schematic-viz/context-menu`

Use this package when the schematic needs a typed, event-driven contextual action surface.

## Key Exports

- `ContextMenuController`
- `ContextMenuManager`
- `MenuItem`
- `ContextMenuState`

## Responsibilities

- resolve menu items against the active target
- maintain visible/hidden menu state
- support keyboard navigation and outside-click dismissal
