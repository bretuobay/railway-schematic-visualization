# SSR API

Package: `@rail-schematic-viz/ssr`

Use this package when you need server-side SVG generation or headless export flows.

## Key Exports

- `SSRRenderer`
- `detectSSREnvironment`
- `setupSSREnvironment`
- `headlessExport`

## Responsibilities

- render SVG without a browser DOM requirement
- report SSR runtime constraints
- provide batch export helpers for server workflows
