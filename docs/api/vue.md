# Vue API

Package: `@rail-schematic-viz/vue`

Use this package when integrating the schematic into a Vue application.

## Key Exports

- `RailSchematicVue`
- `useRailSchematic`

## Responsibilities

- mount and update the schematic in a Vue component tree
- expose composable-driven lifecycle and control actions
- translate shared events into Vue-friendly emits

## Input Contract

- `data` expects a `RailGraph` instance or `null`
