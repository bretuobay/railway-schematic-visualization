# Overlays API

Package: `@rail-schematic-viz/overlays`

Use this package when the base schematic needs annotations, badges, symbols, legends, or other secondary layers.

## Key Exports

- `OverlayManager`
- overlays `SVGRenderer`
- built-in overlay models and legend helpers

## Responsibilities

- register and remove overlays
- manage visibility and ordering
- render overlay markup on top of the base schematic
- support legend and metadata display
