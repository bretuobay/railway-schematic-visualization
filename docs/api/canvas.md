# Canvas API

Package: `@rail-schematic-viz/canvas`

Use this package when large or dense datasets need a canvas-backed render path.

## Key Exports

- `CanvasRenderer`
- `HybridRenderer`

## Responsibilities

- render dense layers with canvas commands
- support hit testing for nodes, edges, and heatmaps
- combine SVG and canvas output in hybrid mode
