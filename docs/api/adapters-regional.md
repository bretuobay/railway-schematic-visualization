# Regional Adapters API

Package: `@rail-schematic-viz/adapters-regional`

Use this package when source data needs to be converted from region-specific formats into `RailGraph`.

## Key Exports

- `CSVAdapter`
- `GeoJSONAdapter`
- `ELRAdapter`
- `RINFAdapter`

## Responsibilities

- parse external data sources
- validate adapter-specific input rules
- emit normalized `RailGraph` instances
