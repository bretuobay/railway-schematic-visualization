# Themes API

Package: `@rail-schematic-viz/themes`

Use this package when you need reusable visual styling rules and built-in theme presets.

## Key Exports

- `ThemeManager`
- `DEFAULT_THEME`
- `DARK_THEME`
- `HIGH_CONTRAST_THEME`
- `COLOR_BLIND_SAFE_THEME`

## Responsibilities

- register and switch themes
- validate theme shape and accessibility constraints
- emit CSS custom properties for host applications
