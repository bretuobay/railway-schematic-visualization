# Plugins API

Package: `@rail-schematic-viz/plugins`

Use this package when you need an extension surface for render hooks, data updates, or viewport events.

## Key Exports

- `PluginManager`
- plugin hook contracts

## Responsibilities

- register and manage plugins
- isolate plugin failures from the main render path
- dispatch lifecycle and rendering hooks
