# Brushing And Linking API

Package: `@rail-schematic-viz/brushing-linking`

Use this package when multiple views need synchronized selection and viewport updates.

## Key Exports

- `BrushingLinkingCoordinator`
- `BrushingLinkingCoordinatorImpl`

## Responsibilities

- keep shared selection state across views
- broadcast viewport changes
- translate coordinates between linked renderers
