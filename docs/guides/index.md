# Guides

These guides are the long-form operational companion to the API reference and Storybook examples. Use them when you are moving from “it renders” to “it ships and stays maintainable.”

## Guide Map

- [Setup Guide](/guides/setup): package selection, installation paths, workspace setup, and first production-safe checks.
- [Framework Integration](/guides/framework-integration): React, Vue, and Web Component integration patterns and adapter selection.
- [Production Rollout](/guides/production-rollout): exports, SSR, canvas, browser compatibility, security, and deployment checks.
- [Migration Playbook](/guides/migration-playbook): phased adoption, upgrades, and rollout-safe migrations.

## Suggested Reading Order

1. Start with [Getting Started](/getting-started) if you are new to the library.
2. Read [Setup Guide](/guides/setup) when the package layout is still being decided.
3. Move to [Framework Integration](/guides/framework-integration) once rendering needs to live inside an app shell.
4. Use [Production Rollout](/guides/production-rollout) before launch.
5. Keep [Migration Playbook](/guides/migration-playbook) for upgrades and phased package adoption.
